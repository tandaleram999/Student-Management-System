# Student Result Management System (SRMS)

A web-based Student Result Management System developed using Java Spring Boot. The application allows administrators to manage student records efficiently through Create, Read, Update, and Delete (CRUD) operations.

## Features

- Add Student
- View Student List
- Update Student Details
- Delete Student
- Responsive Web Interface
- MVC Architecture
- MySQL Database Integration

## Tech Stack

- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
