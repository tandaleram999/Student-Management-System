# 🎓 Student Result Management System (SRMS)

A web-based Student Result Management System developed using **Java Spring Boot**. This application helps administrators manage student records efficiently through CRUD operations.

---

## 📌 Features

- Add New Student
- View Student Details
- Update Student Information
- Delete Student Records
- Responsive User Interface
- MVC Architecture
- MySQL Database Integration
