About SRMS

The Student Result Management System (SRMS) is a web-based application developed using Spring Boot, Thymeleaf,Hibernate, Boostrap, Java, Spring MVC and MySQL.

This system helps administrators efficiently manage student records, including adding, updating, deleting, and viewing student details.

It also allows managing student results, making the entire academic process digital and easy to access.

Key Features:
  • Student Registration
  • Update & Delete Student
  • Result Management
  • Secure Login System
